
# Trader Performance vs Market Sentiment Analysis

## Objective
This project analyzes how Bitcoin market sentiment (Fear & Greed Index)
affects trader profitability and behavior on Hyperliquid.

## Folder Contents
- notebook_1.ipynb : Main analysis notebook
- ds_report.pdf : Final summarized report
- csv_files/ : Processed datasets and summary tables
- outputs/ : Charts and visualizations

## Key Insights
- Extreme Fear has the highest average PnL but low win rate.
- Neutral sentiment shows the most consistent performance.
- Fear and Greed regimes involve larger trade sizes and higher fees.
